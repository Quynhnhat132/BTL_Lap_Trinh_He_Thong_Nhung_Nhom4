##  Hướng dẫn giải nén

File `BTL1.zip` chứa toàn bộ mã nguồn cho bài tập lớn môn Hệ thống nhúng.

### Các bước giải nén:
1. Chuột phải vào file `BTL1.zip`
2. Chọn **Extract Here** hoặc **Extract to BTL1/**
3. Sau khi giải nén sẽ thu được các file mã nguồn (.c, .h)

Lưu ý:  
- Không đổi tên các file `.c` và `.h`  
- Giữ nguyên cấu trúc thư mục khi import vào e² studio
